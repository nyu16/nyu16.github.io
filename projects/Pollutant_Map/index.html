Washington Environmental Pollutant Map

A three-dimensional map of pollutants in Washington State census tracts

Go to Map

Seattle, 2017 - Thom Milkovic

Map Description:

This website hosts an interactive web map that shows the values of environmental pollutant variables in each Washington state census tract. The three-dimensional map represents the values of a given pollutant in a census tract through both the height and color of each tract’s representing polygon. The pollutant currently shown in the map can be identified with the legend on the side of the map and changed by clicking on the accompanying sidebar. Further information on a pollutant's value in a given census tract can be found by clicking on that tract.

The map in this website was made for educational purposes only and is intended to bring attention to the distribution of pollutants in Washington state, and the possible inequalities associated with that distribution.

This map was created as part of a final project in the Winter 2023 Geography 458: Advanced Digital Geographies class taught by Professor Bo Zhao at the University of Washington, Seattle. Further information on the authors, contributors, and acknowledgments can be found on the about us page.

Data:

The data used in this map was gathered from the environmental health disparities section of the Washington Geospatial Open Data Portal. The variables contained in the data were normalized using min-max normalization, to allow for easier comparison between different pollutants. Further information about the data, variables, and treatment can be found on the data page.
